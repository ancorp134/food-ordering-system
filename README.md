## Roles

1. ADMIN  
   - Manage users and system-level configs

2. RESTAURANT_OWNER  
   - Create & manage own restaurants and menus

3. CUSTOMER  
   - Browse restaurants, place orders

4. DELIVERY_AGENT  
   - Accept and deliver orders

## Core Flow

1. User registers & logs in
2. Customer browses restaurants
3. Customer views menu items
4. Customer adds items to cart
5. Customer places order
6. Payment is initiated
7. Order status is updated
8. Notification is sent
